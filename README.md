# ORI-N-PRO
ORIÉN nace en mayo de 2026 como una empresa filial de RSI, bajo la visión estratégica del Ing. Sergio. La compañía surge para liderar el sector multimedia de alta gama, especializándose en soluciones integrales de audio y video profesional, con tecnología de vanguardia como paneles de píxel pitch 2.9 (P2.9).
